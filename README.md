# automatic-werewolf-offline-host
Đây là ứng dụng web quản trò Ma Sói real-time cho các trận đấu offline. Hệ thống thay thế quản trò con người bằng cách dùng 1 máy chủ phát âm thanh/nhạc nền điều hướng, kết hợp WebSockets đồng bộ đến điện thoại từng người chơi. Máy con tự động sáng/tối theo role để tương tác bí mật ban đêm. Backend Python xử lý logic và check win tự động.
